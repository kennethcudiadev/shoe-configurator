# 3D Shoe Model

Place your `shoe.glb` file here. The model should have:

- **Nodes**: `Upper`, `Sole`, `Laces` (mesh geometry)
- **Materials**: `Upper_Mat`, `Sole_Mat`, `Laces_Mat`

You can find free shoe models at [Sketchfab](https://sketchfab.com) or [CGTrader](https://www.cgtrader.com). If your model has different node/material names, you may need to adjust `src/three/Shoe.tsx`.

If the model is missing, a placeholder geometry will be shown instead.
